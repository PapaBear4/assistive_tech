# Assistive Tech App (Phineas)

A Flutter application for assistive technology, focusing on document annotation and audio feedback.

## The Bitmap Strategy
This app uses a unique approach called "The Bitmap Strategy" where imported documents (PDFs/Images) are converted into static PNG images. Users then overlay text annotations on top of these images.

## Getting Started

1.  **Prerequisites**: Ensure you have Flutter installed and set up.
2.  **Dependencies**: Run `flutter pub get` to install dependencies.
3.  **Code Generation**: Run `dart run build_runner build` to generate Drift and Riverpod code.
4.  **Run**: Use `flutter run` to start the application.
